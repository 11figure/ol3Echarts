# ol3Echarts

> 基于openlayers3+新版扩展的echarts3的图表插件，暂时支持echarts的所有map组件类型，对普通不支持坐标系的图表正在兼容（饼图，柱状图，折线图已兼容）

## 下载

```bash
git clone https://github.com/sakitam-fdd/HMap.git
npm install
npm run build
```
